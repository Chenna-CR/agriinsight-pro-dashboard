# 🌾 AgriInsight Pro – Crop Intelligence Dashboard

## 📊 Overview
AgriInsight Pro is a modern Power BI dashboard designed to analyze crop production, yield, revenue, and environmental factors across multiple Indian states.

---

## 🚀 Features
- 📈 Multi-year production trend with moving average
- 🌦 Rainfall vs Yield correlation analysis
- 🌍 State-wise performance insights
- 🌱 Crop-wise production and revenue analysis
- 📊 Interactive filters (State, Season, Crop, Year)
- 🎨 Modern UI with custom design elements

---

## 📁 Dataset
- 16,000+ records
- 13 States
- 14 Crops
- Years: 2005–2022

---

## 📌 Key Insights
- Identifies high-performing crops and regions
- Shows impact of rainfall on yield
- Helps in agricultural decision-making

---

## 🛠 Tools Used
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
- Custom UI Design
